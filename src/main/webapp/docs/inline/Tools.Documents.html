Tools.Documents()

new Documents()

The Documents tool shows a table of the documents in the corpus and includes functionality for modifying the corpus.

Tutorials:
  • Documents
Example
let config = {
    "columns": null,
    "dir": null,
    "docId": null,
    "docIndex": null,
    "query": null,
    "sort": null,
  };

  loadCorpus("austen").tool("documents", config);
