Tools.RezoViz()

new RezoViz()

RezoViz represents connections between people, places and organizations that co-occur in multiple documents.

Tutorials:
  • RezoViz
Example
let config = {
	"docId": null,
	"limit": null,
	"minEdgeCount": null,
	"nerService": null,
	"type": null,
};

loadCorpus("austen").tool("rezoviz", config);
