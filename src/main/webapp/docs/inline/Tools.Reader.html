Tools.Reader()

new Reader()

The Reader tool provides a way of reading documents in the corpus, text is fetched as needed. You can work with corpus text programmatically using Spyral.Corpus#texts.

Tutorials:
  • Reader
Example
let config = {
    "limit": null,
    "query": null,
    "skipTodocId": null,
    "start": null
  };

  loadCorpus("austen").tool("Reader", config);
